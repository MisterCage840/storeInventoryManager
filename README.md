Inventory Management Application

A full-stack Inventory Management System built with Node.js, Express, PostgreSQL, and EJS.
The app allows managing items, categories, and stock levels, and is deployed using Render (backend) and Neon (PostgreSQL).

Tech Stack

Node.js, Express

PostgreSQL (Neon)

EJS, HTML, CSS, Vanilla JS

Render (Hosting)

Features

CRUD operations for inventory items

Category management

Stock quantity tracking with low-stock visual indicators

Secure delete actions with password protection

Server-side rendering with EJS

MVC architecture
